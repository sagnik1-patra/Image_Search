##  API
[Unsplash API](https://unsplash.com/documentation)

## Run Locally
1. Clone repo
2. Replace `YOUR_ACCESS_KEY` in `script.js` with your Unsplash API key.
3. Open `index.html` in browser or deploy to GitHub Pages/Netlify.
